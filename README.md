# ShopMini - Mini E-Commerce Platform

A full-stack e-commerce web application built with the MERN stack (MongoDB, Express, React, Node.js) for Flipkart Grid 8.0.

## Features
- Product listing with responsive grid layout
- Shopping cart with quantity management
- Checkout flow with order form
- RESTful API for product management
- MongoDB Atlas cloud database integration

## Tech Stack
Frontend: React (Vite), React Router, Context API, Axios
Backend: Node.js, Express.js, Mongoose
Database: MongoDB Atlas
